列表

  • 序列是Python中最基本的数据结构。序列中的每个元素都分配一个数字–索引，从0开始
  • Python中6个内置的序列类型
    • 字符串（string）
    • 列表（list）
    • 元组（tuple）
    • 字节串（bytes）
    • 字节数组（bytearray）
    • 范围（range）
  • 序列操作
    • 索引
    • 切片
    • 加乘
    • 检查成员
    • 长度
    • 最大最小
  • 列表
    • 方括号**[]**内逗号分隔
    • 列表的数据项类型可以不同

1 列表操作

访问列表元素

  • 通过下标索引访问

    list1 = ['physics', 'chemistry', 1997, 2000]
list2 = [1, 2, 3, 4, 5, 6, 7 ]
 
print("list1[0]: ", list1[0])
print("list2[1:5]: ", list2[1:5])
#list1[0]:  physics
#list2[1:5]:  [2, 3, 4, 5]

列表更新

  • del删除列表元素：del list[index]
  • 列表修改：list[index]=newvalue

列表操作符

  • Python 表达式 结果 描述
    len([1, 2, 3]) 3 长度
    [1, 2, 3] + [4, 5, 6] [1, 2, 3, 4, 5, 6] 组合
    [‘Hi!’] * 4 [‘Hi!’, ‘Hi!’, ‘Hi!’, ‘Hi!’] 重复
    3 in [1, 2, 3] True 元素是否存在于列表中
    for x in [1, 2, 3]: print x, 1 2 3 迭代

列表比较

# 导入 operator 模块
import operator

a = [1, 2]
b = [2, 3]
c = [2, 3]
print("operator.eq(a,b): ", operator.eq(a,b))
print("operator.eq(c,b): ", operator.eq(c,b))

operator.eq(a,b):  False
operator.eq(c,b):  True

2 列表函数&方法

函数&方法 描述
cmp(list1, list2) 比较两个列表的元素（Python 3.X 的版本移除）
len(list) 列表元素个数
max(list) 返回列表元素最大值
min(list) 返回列表元素最小值
list(seq) 将元组转换为列表
[sorted(list,reverse=False)][] 临时排序，返回一个新列表.print(sorted(list))
list.append(obj) 在列表末尾添加新的对象，无返回值
list.count(obj) 统计某个元素在列表中出现的次数
list.extend(seq) 在列表末尾一次性追加另一个序列中的多个值（用新列表扩展原来的列表）
[list.index(x, start[, end]]) 从列表中找出某个值第一个匹配项的索引位置
list.insert(index, obj) 将对象插入列表，无返回值
list.pop([index=-1]) 移除列表中的一个元素（默认最后一个元素），并且返回该元素的值
list.remove(obj) 移除列表中某个值的第一个匹配项，无返回值
list.reverse() 反向列表中元素，无返回值
list.sort(cmp=None, key=None, reverse=False) 对原列表进行排序，key：比较对象，reverse = False 升序，永久性操作

3 遍历列表

for循环遍历

  • 索引遍历

    for idx in range(len(list)):
    print(list[idx])

  • 元素遍历

    for item in list:
    print(item)

  • 列表表达式

    matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
flattened = [num for row in matrix for num in row]  # 创建一个包含matrix中所有数字的列表

  • 列表解析

    >>> data = [ 'ACME', 50, 91.1, (2012, 12, 21) ]
>>> _, shares, price, _ = data  # 通过下划线来丢弃其他值，可以使用认识变量名
>>> shares
50
>>> price
91.1

# 也可解压多个元素
>>> name,*numbers,date = data
>>> numbers
[50,91.1]

4 复制列表

  • ==直接变量赋值==：同一存储地址，同步更新，相当于一个别名

    list1 = [1,2,3,4]
list2 = list1
print(id(list1) == id(list2))
# True

  • ==切片复制==：不同存储地址

    list1 = [1,2,3,4]
list2 = list1[:]
print(id(list1) == id(list2))
# False

  • ==copy方法==：不同存储地址

    list1 = [1,2,3,4]
list2 = list1.copy()
print(id(list1) == id(list2))
# False

  • ==list方法==：不同存储地址

    list1 = [1,2,3,4]
list2 = list(list1)
print(id(list1) == id(list2))
# False

  • ==for循环==：通过遍历添加元素

    list1 = [1,2,3,4]
list2 = []
for item in list1:
    list2.append(item)

  • ==列表推导式==：不同存储地址

    list1 = [1,2,3,4]
list2 = [item for item in list1]

5 数值列表–range

  • range函数用于创建整数列表，常用于for循环

  • 使用：range(start,stop,step)

    • start:开始数值，默认为0

    • stop:结束数值，不包括stop

    • step:步长，默认为1，可以为负，表示递减

    • 生成的数值列表范围：start<= value(+step) < stop (左闭右开)

    num_list = list(range(5,0,-1))
print(num_list)
[5,4,3,2,1]
